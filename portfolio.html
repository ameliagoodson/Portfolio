What's in your fridge?

What's in your Fridge is a recipe search app with a difference: the user searches by ingredient. This web app is designed so that people can search recipes by the ingredients they already have, to prevent them from having to go grocery shopping, to minimise waste and to encourage creative cooking.

Github Website

What's the weather like?

Interactive weather app featuring current and forecasted weather conditions including temperature, humidity, wind speed and UV index. The user's search history is saved via local storage to the sidebar so they can access their saved searches.

Github Website

Daily planner

This daily planner features timeblocks for standard business hours, which are shaded different colours based on the time of day using the moment.js framework. Users can enter the details of their schedule in the planner and press the 'save' button, which saves the information to local storage and remains constant upon page refresh.

Github Website

Google News search

Google News search app that allows users to search by keyword for local news or all related articles. Users can select number of articles to be displayed and filter by language or country.
